## 📋 项目简介

时间银行是一款创新的Android时间管理应用，通过"时间兑换"机制帮助用户培养良好的手机使用习惯。用户通过使用正向应用（学习、阅读类）积累时间信用，用于解锁负向应用（娱乐、游戏类）的使用权限。

### 核心特性
- ✅ **离线运行** - 完全本地化，不联网，保护隐私
- ✅ **时间兑换** - 正向激励机制，培养良好习惯
- ✅ **智能拦截** - 余额不足时自动拦截负向应用
- ✅ **数据可视** - 直观的图表展示使用趋势
- ✅ **定期重置** - 每周清零，开始新的挑战

## 🎯 项目目标

## 🎨 核心功能概览

### 1. 应用分类管理
- 将应用分为正向、负向、未分类三类
- 支持搜索和批量操作
- 提供常见应用的预设分类

### 2. 时间统计与积累
- 实时监控应用使用时长（精度：秒级）
- 正向应用使用时长自动转换为信用额度
- 负向应用使用时消耗信用额度

### 3. 负向应用拦截
- 余额不足时自动拦截负向应用
- 全屏拦截页面，延迟5秒后可返回
- 显示激励文案和当前余额

### 4. 数据可视化
- 主页仪表板显示余额和今日统计
- 使用记录页面展示趋势图表
- 应用使用排行榜

### 5. 定期重置
- 自定义重置周期（默认每周一）
- 自动清空统计数据和余额
- 每周允许手动重置1次
